# chelyabinsk
R package for geochemical data plot

# Installation

    R> library(devtools)
    R> install_github('misasa/chelyabinsk')

# Developer's guide

Generate documentation

```
$ cd ~/devel-godigo/R/chelyabinsk
$ Rscript -e "devtools::document()"
```

Run test

```
$ R CMD check ../chelyabinsk
```

Push to the Git server
